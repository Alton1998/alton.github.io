# LangChain
## Overview
Consists of 3 components:

- Components:
  - LLM Wrappers
  - Prompt Templates
  - Indexes for information Retrieval
- Chains: Assemble Components to solve a specific task
- Agents: allows LLMs To interact with it's environment
